"c [se] \u00e4r den tredje bokstaven i det moderna latinska alfabetet.\n\n\n== uttal och anv\u00e4ndning ==\nc uttalas normalt som s framf\u00f6r mjuk vokal (e, i, y, \u00e4 och \u00f6) och som k framf\u00f6r h\u00e5rd vokal (a, o, u och \u00e5) och konsonant. i svenskan anv\u00e4nds bokstaven c dock normalt endast i ord med s-ljud och i dubbelteckning av k-ljudet som \"ck\", medan ord med k-ljud d\u00e4remot normalt stavas med bokstaven k som exempelvis orden kamera, kanon och karavan, till skillnad fr\u00e5n flera andra spr\u00e5k som normalt stavar ord med k-ljud med c, \u00e4ven om det finns undantag d\u00e4r c beh\u00e5lls, som exempelvis orden camping och cancer. ett specialfall \u00e4r \"caesar\", d\u00e4r a i detta fall \u00e4r stumt i svenskan, vilket f\u00f6r uttalet inneb\u00e4r att c f\u00f6reg\u00e5r den mjuka vokalen e och s\u00e5ledes uttalas som s. d\u00e4remot skrives c som cesar i bokstaveringssammanhang, detta m\u00f6jligtvis f\u00f6r tydligheten i uttalet.uttalsreglerna av c liknar de svenska i m\u00e5nga spr\u00e5k som anv\u00e4nder det latinska alfabetet, det vill s\u00e4ga uttalet styrs av efterf\u00f6ljande ljud, men i klassiskt latin uttalades c alltid som k.\nbokstavskombinationen \"ch\" kan uttalas med sj-ljudet, exempelvis i \"chock\" och \"choklad\", eller tj-ljudet, exempelvis i \"check\". c ing\u00e5r ocks\u00e5 i bokstavskombinationen \"sch\", vilken uttalas som sj-ljud exempelvis i \"schimpans\" eller \"sch\u00e4fer\".\n\n\n== betydelser ==\n\n\n=== versalt c ===\nversalt c kan vara:\n\nett av de vanligaste programspr\u00e5ken. se c (programspr\u00e5k), c++ samt c#\nl\u00e4nsbokstav f\u00f6r uppsala l\u00e4n.\nsymbol f\u00f6r talet 12 i det hexadecimala talsystemet.\nf\u00f6rkortning f\u00f6r coulomb, si-enheten f\u00f6r elektrisk laddning.\ni fysiken symbol f\u00f6r storheten kapacitans.\ngrundtonen i den rena c-durskalan (spelas p\u00e5 piano med enbart de vita tangenterna). ettstrukna c har frekvensen 261,626 hertz.\nkemiskt tecken f\u00f6r grund\u00e4mnet kol. se \u00e4ven periodiska systemet.\ninom bibliotekens klassifikationssystem sab beteckning f\u00f6r religion, se sab:c.\nflera betydelser inom matematiken, bland annat komplexa talplanet.\ni det romerska talsystemet beteckning f\u00f6r talet 100, lat; centum.\nnationalitetsbeteckning f\u00f6r motorfordon fr\u00e5n kuba.\nbetyget underk\u00e4nd i en \u00e4ldre betygsskala.\nen klass torpedb\u00e5tar i den finl\u00e4ndska marinen, se till exempel c 1 (torpedb\u00e5t, 1902).\nbeteckning f\u00f6r centerpartiet - (c).\nbeteckning f\u00f6r det politiska partiet centern i finland.\ninfluensavirus typ c.\ngrad celsius (\u00b0c), enhet f\u00f6r temperatur.\ns\u00f6ndagsbokstav f\u00f6r normal\u00e5r som b\u00f6rjar en fredag\nf\u00f6rkortning f\u00f6r centrum eller centralstation\n\n\n=== gement c ===\nbeteckning f\u00f6r ljusets hastighet (i vakuum).\nbeteckning f\u00f6r kardinaltalet f\u00f6r de reella talen, kontinuum. c = 2alef-0\nbeteckning f\u00f6r flera myntsorter, bland annat cent och centime.\ninom kemi beteckning f\u00f6r koncentrationen i en kemisk l\u00f6sning.\nf\u00f6rkortning f\u00f6r prefixet centi.\n\n\n== historia ==\ntill det latinska alfabetet kom bokstaven c fr\u00e5n det grekiska alfabetets gamma via etruskernas alfabet, som anv\u00e4nde en variant av det grekiska alfabetet. grekerna hade i sin tur f\u00e5tt sin bokstav \"gamma\" fr\u00e5n feniciernas bokstav \"gimel\". den kan h\u00e4rstamma fr\u00e5n en egyptisk hieroglyf f\u00f6r en stavslunga (namnet p\u00e5 bokstaven ska d\u00e5 ocks\u00e5 ha kommit fr\u00e5n namnet p\u00e5 vapnet), eller s\u00e5 f\u00f6rest\u00e4llde bokstaven ursprungligen en kamel, eller snarare ett kamelhuvud (djuret hette \"gamal\" p\u00e5 gammal semitiska). eftersom det etruskiska spr\u00e5ket inte hade n\u00e5gra tonande klusiler (/b/, /d/ och /g/), s\u00e5 kom den h\u00e4r bokstaven att anv\u00e4ndas f\u00f6r att skriva deras k-ljud. n\u00e4r romarna sedan \u00f6vertog bokstaven anv\u00e4ndes den troligtvis i b\u00f6rjan b\u00e5de f\u00f6r /k/ och /g/, och f\u00f6rst senare skapades bokstaven g  genom att l\u00e4gga till ett v\u00e5gr\u00e4tt streck i h\u00f6gerkanten p\u00e5 c:et f\u00f6r att uttrycka /g/. m\u00f6jligtvis anv\u00e4ndes c ursprungligen bara f\u00f6r g-ljudet (och k f\u00f6r k-ljudet).\n\n\n== datateknik ==\ni datorer lagras c samt f\u00f6rkomponerade bokst\u00e4ver med c som bas och vissa andra varianter av c med f\u00f6ljande kodpunkter:\n\ni ascii-baserade kodningar (hexadecimalt).\n\u00f6vriga varianter av c lagras med olika v\u00e4rden beroende p\u00e5 vilken kodning som anv\u00e4nds, om de alls kan representeras.\n\n\n== referenser =="
