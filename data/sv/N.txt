"n [enn] \u00e4r den fjortonde bokstaven i det moderna latinska alfabetet.\nd\u00e5 man vill s\u00e4kerst\u00e4lla att n\u00e5gon uppfattat bokstaven r\u00e4tt vid bokstavering uttalar man ofta namnet niklas.\n\n\n== betydelser ==\n\n\n=== versalt n ===\nl\u00e4nsbokstav f\u00f6r hallands l\u00e4n.\nkemiskt tecken f\u00f6r grund\u00e4mnet kv\u00e4ve. se \u00e4ven periodiska systemet.\nf\u00f6rkortning f\u00f6r n\u00e4ringsdepartementet.\ntalet nittio (90) i det romerska talsystemet.\nf\u00f6rkortning f\u00f6r newton, enheten f\u00f6r kraft i internationella m\u00e5ttenhetssystemet.\nnationalitetsbeteckning f\u00f6r motorfordon fr\u00e5n norge.\ninom bibliotekens klassifikationssystem sab beteckning f\u00f6r \u00e4mnet geografi, se n (sab).\nen kontakttyp f\u00f6r koaxialkabel, se n-kontakt.\n\u2115 betecknar i matematik m\u00e4ngden av de naturliga talen.\nf\u00f6rkortning f\u00f6r norr.\n\n\n=== gement n ===\nsymbol f\u00f6r nano, si-prefix f\u00f6r faktorn 10\u20139.\ngrundstorhet som anger substansm\u00e4ngd inom kemi och fysik.\nn betecknar i matematik ett numeriskt tal med ok\u00e4nt eller valfritt talv\u00e4rde.\n\n\n== historia ==\ntill det latinska alfabetet kom bokstaven n fr\u00e5n den grekiska bokstaven ny, som i sin tur h\u00e4rstammade fr\u00e5n den feniciska bokstaven , \"nun\". den f\u00f6rest\u00e4llde ursprungligen en orm (\"nahsh\"), men fick sedan namnet \"nun\", som egentligen betyder \"fisk\".\n\n\n== datateknik ==\ni datorer lagras n samt f\u00f6rkomponerade bokst\u00e4ver med n som bas och vissa andra varianter av n med f\u00f6ljande kodpunkter:\n\ni ascii-baserade kodningar lagras n med v\u00e4rdet 0x4e (hexadecimalt) och n med v\u00e4rdet 0x6e (hexadecimalt).\ni ebcdic-baserade kodningar lagras n med v\u00e4rdet 0xd5 (hexadecimalt) och n med v\u00e4rdet 0x95 (hexadecimalt).\n\u00f6vriga varianter av n lagras med olika v\u00e4rden beroende p\u00e5 vilken kodning som anv\u00e4nds, om de alls kan representeras."
